## Project Overview
This project focuses on analyzing Formula One race performance data using Python, Pandas, SQL, and data visualization techniques to evaluate driver consistency, tyre degradation, race pace trends, and strategy effectiveness.

The analysis was built to better understand how data-driven decision-making impacts race outcomes in high-performance motorsport environments.

---

## Why I Built This Project
I built this project to combine my passion for Formula One with data analytics and explore how teams use performance insights, strategy evaluation, and operational analysis to improve competitiveness during a race weekend.

---

## Business & Engineering Thinking
In Formula One, race strategy is not only an engineering challenge but also a business-critical decision-making process where milliseconds, tyre performance, pit stop timing, and consistency directly influence race outcomes and team performance.

This project applies analytical thinking to simulate how data can support:
- performance optimization
- strategic decision-making
- operational efficiency
- competitive advantage in high-pressure environments
