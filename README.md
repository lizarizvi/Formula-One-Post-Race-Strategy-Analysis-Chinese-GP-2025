# Formula One Post-Race Strategy Analysis – Chinese GP 2025

## Project Overview
This project analyzes Formula One race performance and strategy trends from the Chinese Grand Prix 2025 using Python, Pandas, SQL, and data visualization techniques.

## Key Analysis Areas
- Driver consistency
- Tyre degradation
- Race pace analysis
- Strategy effectiveness
- Performance comparison
- Lap 

## Tools Used
- Python
- Pandas
- SQL
- Google Colab
- Data Visualization
